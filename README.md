# HealthSync

HealthSync is an appointment management application consisting of a Java Spring Boot backend and a React frontend.

## 🚀 Running the Application on Windows

### 💻 Backend Setup & Run

The backend is built with Spring Boot 3.x and requires **Java 17 or higher**.

Navigate to the server directory and start the application using the Maven Wrapper (`mvnw.cmd`):

```powershell
cd D:\project\healthsync\server
.\mvnw.cmd spring-boot:run
```

### 🌐 Frontend Setup & Run

The frontend is built with React and Vite.

Navigate to the client directory, install the required packages, and run the development server:

```powershell
cd D:\project\healthsync\client
npm install
npm run dev
```
